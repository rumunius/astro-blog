# T1 求一维数组中的最小元素（填空题）

## 题干

找到$n$个整数中最小的那个，求它的值$v$和它的下标$k$（从0开始计数）.

## 输入格式

第一行输入整数$n$，接下来输入$n$个整数，数据之间都用一个空格间隔开来。

## 输出格式

第一行输出$v$和$k$，中间用一个空格隔开。
## 输入输出样例

### 输入
```
9 9 8 7 3 2 1 4 5 6
```
### 输出
```
1 5
```
# T2 斐波那契数列
## 题干

输入整数$n$，然后：

 - 若$n$非正数，输出`error!`后接一个回车。
 - 若$n>100$，输出`The number is too large!`后接一个回车。
 - 否则，按原顺序输出以$1$开头的斐波那契数列$1,1,2,3,5,8,\cdots$（`a[i]=a[i-1]+a[i-2]`）中不大于$n$的那些数字，用`,`隔开，之后是一个空格，之后是这些数的平均值（保留一位小数）
## 输入格式

第一行输入整数$n$。
## 输出格式

见题干。
## 输入输出样例

### 输入
```
10
```
### 输出
```
1,1,2,3,5,8 3.3
```
# T3 字符串插入（编写一个函数）
## 题干

编写函数`void interleave_strings(char *s,char *t,char *r)`，其功能为：

- 将s的第一个字符作为r的第一个字符，之后依次在r的尾部添加：t的第一个字符，s的第二个字符，t的第二个字符……
- 如果s,r长度不等，在较短的字符串耗尽之后将较长字符串的剩余部分接到r的尾部。
- 如果s,r中有一个是空指针，则r是一个单字符字符串`*`
## 传参样例

### 传入
```
s="abc"
t="12345"
```
### 返回
```
r="a1b2c345"
```

## 数据范围

$$0<s,t\text{{}}<50$$
# T4 回文质数

## 题干

回文质数是指既是一个质数又是一个回文数（从左到右和从右到左是看一样的）的数，例如$151$。

写一个程序来找出范围 $[a,b] (5 \le a < b \le 100000)$间的所有回文质数。

## 输入格式

第一行输入两个正整数 $a$ 和 $b$。

## 输出格式

输出一个回文质数的列表，用一个空格间隔开来，注意，输出的末尾不是一个空格而是一个换行。

## 输入输出样例

### 输入

```
5 300
```

### 输出

```
5 7 11 101 131 151 181 191

```
